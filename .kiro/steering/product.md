# Product Overview

LvlUp is a gamified knowledge management system that transforms skill acquisition into an RPG-like experience. Users track their developer growth through an interactive skill tree, earn achievements, and generate AI-powered resumes.

## Core Features

- **Gamification System**: XP, levels, skill points, rank progression (Bronze → Master)
- **Interactive Skill Tree**: Node-based visualization with parent-child dependencies and tier progression
- **Achievement Engine**: Badge system with rarity tiers (Common → Mythic) and automatic triggers
- **AI Resume Builder**: Smart matching of projects to job descriptions with keyword analysis
- **Daily Rewards**: Login streak system with escalating rewards and gacha currency (Primogems)

## Design Philosophy

Inspired by Genshin Impact and Honkai Star Rail aesthetics:
- Animated gradient backgrounds with particle effects
- Star-field backdrops and glow effects for rarity tiers
- Smooth transitions and hover effects
- Custom fonts: Orbitron (display) + Rajdhani (body)

## Target Users

Developers and students building portfolios who want to:
- Track skill progression visually
- Maintain motivation through gamification
- Generate tailored resumes for job applications
- Showcase projects in an engaging format
